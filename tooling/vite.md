# Vite

```bash
# scaffold a Vite project
# walk through the options
pnpm create vite
yarn create vite
pnpm create vite
```

## Running Vite

```bash
# run Vite with PNPM
pnpm run dev # or, `npm run dev` when using NPM

# or, you can also just run Vite
vite
# or, `vite dev`
# or, `vite serve`
```

## Build project

```bash
npm run build
pnpm run build
```
