# Node Package Manager

## npm commands

```bash
# Initialize project folder
npm init # creates a package.json

# List all config settings
npm config list

# List installed packages
npm list # local
npm list -g # global

# Uninstall a package
npm uninstall <package>

# Find outdated packages l/g
npm outdated
npm outdated -g

# Update
npm update # all local outdated packages
npm update <package> # global packages
```
